Поисковый фильтр

Опция, которая особенно понравилась пользователям. Сейчас можно посмотреть доступные дела по фильтрам:

  • «Все дела» — все актуальные запросы
  • «Дела для меня» — запросы, сформированные на основании навыков и местоположения пользователя
  • «Мой фильтр» — выбор параметров, по которым пользователь может установить свои критерии поиска.
